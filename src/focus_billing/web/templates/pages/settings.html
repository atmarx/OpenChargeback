Review Flag Patterns

Charges matching any of these patterns (in service name, resource ID, or resource name) will be flagged for review. Uses Python regex syntax.

Fund/org codes that don't match ANY of these patterns will be flagged for review. Leave empty to skip validation.

Regex Pattern Help

Sample Data Patterns

Examples matching your sample data format:

.*gpu.* Flag GPU instances for review
.*training.* Flag ML training resources
^[A-Z]+-.*-\d{4}$ Fund/org: PREFIX-xxx-YEAR
^(NIH|NSF|NASA)- Federal grant prefixes

Common Patterns

.*keyword.* Contains "keyword" anywhere
^PREFIX- Starts with "PREFIX-"
-suffix$ Ends with "-suffix"
^\d{6}$ Exactly 6 digits

Special Characters

. Any character
.* Any characters (0+)
\d Any digit
\w Any word character
^ Start of string
$ End of string
💡
Patterns are case-insensitive. They are applied during CSV import (CLI or web).
