# User Guide

This section covers day-to-day usage of OpenChargeback for both CLI and web interface users.

## Contents

1. [CLI Reference](cli.md) - Command-line interface for scripting and automation
2. [Web UI Guide](web-ui.md) - Browser-based interface for interactive use
3. [FOCUS Format](focus-format.md) - Understanding the billing data format

## Which Interface Should I Use?

| Use Case | Recommended |
|----------|-------------|
| Automated imports from scripts | CLI |
| Manual imports and review | Web UI |
| CI/CD pipeline integration | CLI |
| Generating statements interactively | Web UI |
| One-off data corrections | Web UI |
| Bulk operations | CLI |

Both interfaces use the same underlying database, so you can mix and match as needed.
